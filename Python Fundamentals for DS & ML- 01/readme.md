## 🧪 Sample Code Snippets

```python
# Numeric Types
age = 25
height = 5.9
print("Age:", age, "Type:", type(age))
print("Height:", height, "Type:", type(height))

# String Concatenation
name = "Rossi"
greeting = 'Hello, ' + name
print(greeting)

# Boolean and Conditional
is_student = True
if is_student:
    print("Student detected!")
```
